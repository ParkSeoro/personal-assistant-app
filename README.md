# 쟈비스 AI 비서 앱

본 프로젝트는 웹 기반 개인 비서 애플리케이션의 예제입니다. 음성 입력/출력, 간단한 데이터 보안, 위치 권한 확인 기능 등을 포함합니다. 모바일 웹이나 하이브리드 앱(WebView)을 통해 안드로이드 기기에서 사용할 수 있습니다.

## 주요 기능
- **대화형 채팅 인터페이스**: 텍스트 입력 또는 음성 인식을 통해 비서와 대화합니다.
- **음성 지원**: 웹 스피치 API를 활용하여 음성 입력(SpeechRecognition)과 음성 출력(SpeechSynthesis)을 제공합니다.
- **데이터 보안**: 채팅 기록은 브라우저 `localStorage`에 AES-GCM 방식으로 암호화되어 저장됩니다.
- **위젯**: 일정 관리, 감정 분석, 인사이트 표시 등의 간단한 위젯 예제가 포함되어 있습니다.
- **권한 확인**: 마이크 및 위치 접근 권한 상태를 확인하여 모바일 권한 관리에 활용할 수 있습니다.

## 실행 방법
1. 저장소를 클론한 후 `index.html`을 브라우저로 열어 실행합니다.
2. 마이크 사용을 허용하면 음성 입력이 가능하며, 안드로이드 기기에서는 위치 권한을 요청하여 현재 위치 정보를 확인할 수 있습니다.

## 제안 기능
- **모바일 앱 패키징**: Cordova나 Capacitor와 같은 프레임워크를 사용하면 안드로이드 네이티브 기능(주소록, 알림 등)에 접근할 수 있습니다.
- **클라우드 동기화**: 백엔드 서버를 추가해 여러 기기 간 채팅 기록과 일정 데이터를 안전하게 동기화합니다.
- **개인화 기능**: 사용자의 패턴을 학습하여 맞춤형 알림이나 추천을 제공합니다.



## 확장 설계 문서
- 상세 아키텍처와 단계별 구현 계획은 `ROADMAP_KO.md`를 참고하세요.

- **엔진 상태/정책 패널**: 멀티 엔진 상태 표시, 고위험 작업 승인 정책, 자동 투자/메시지 분석 권한 토글을 제공합니다.
- **활동 로그 타임라인**: 엔진별 작업 내역을 시간순으로 확인할 수 있습니다.
